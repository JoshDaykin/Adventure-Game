# Adventure Game (C++)

## Overview 
A C++ text-based game implementing saving, inventory and stealth mechanics.
Built to practice object-oriented programming and file I/O.

## Features
- Player classes and stats
- Save and load functionality using file I/O
- Inventory system with potion usage
- Sneak and speech mechanics based on player statistics
- Basic combat and progression system

## Technologies Used
- C++
- Visual Studio
- File Input/output
- Object-oriented programming principles

## What I learned
- Structuring larger C++ programs
- Managing persistent data using files
- Desigining systems that are stable and reusable
- Debugging and testing gameplay logic
